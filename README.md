#  Bon appétit – Food Ordering App

## Description

Bon appétit is a modern food ordering app that allows users to explore menus, order their favorite meals, and enjoy a smooth and convenient delivery experience.

## Live URL

https://mern-food-ordering-app-frontend-cwu8.onrender.com/

## Backend Repository

https://github.com/Jhalakmahansaria123/mern-food-ordering-app-backend

## Tech Stack

Frameworks & Libraries: React, Shadcn UI  
Backend: Node.js  
Database: MongoDB  
Authentication: Auth0  
Payments: Stripe  
Media Storage: Cloudinary  
Deployment: Render  

##  Features

-  Secure user authentication using Auth0 (login & signup)  
-  Test-based payment integration using Stripe  
- Use test card number: **4242 4242 4242 4242**  
-  Search for food across cities like **Kolkata, Mumbai, London**, etc.  
-  Real-time order tracking  
- Browse menus and place orders easily  

## How to Use

1. Open the live app using the link above  
2. Sign up or log in using your account  
3. Search for your desired city or food items  
4. Add items to your cart and proceed to checkout  
5. Use the Stripe test card to simulate payment  
6. Track your order in real time
